# magicWeather 梦幻天气
#
### @写在开头
### 小程序日渐流行的2018年，学习了小程序的开发，以天气为练手主题写了一个查看全国省市天气预报的小程序。
### 做的比较简单，练手为主，发现小程序的可玩性特别强，API功能非常丰富，也有一点无脑。
